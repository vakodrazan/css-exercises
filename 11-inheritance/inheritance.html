CSS inheritance

What is CSS inheritance?

When we heard of CSS inheritance the first things come to our mind is that children elements are inheritated their parents elements.

The properties that have inherit value.

There are some properties that have inherit value by default. These are some of them: font-weight, font-style, line-height and text-align.

Difference between the currentcolor and inherit

  • currentcolor sets the spicification of the color but it is for the independant elements.
  • inherit sets the specification of color. It is child elements which inheritated its parents.

The use of revert, initial and unset value

revert is used to put back the things that you have been changed into its origin.

initial seems as if there the user don't want to use any style but it sets from the browser.

unset is used to unstaged style and you dont want to apply them.

Any dangers or unitended side-effects

Be careful when using inherit value because it is sometimes changed things thatt you don't mean to.

And the same as the other value like revert, initial and unset you need to emphacise the elements you want because it can affect all your work.

The possibilities with using an ancestor or universal selector.

It can be helpful to use ancestor or universal selector , combined with inherit value because it makes the elements more important.
