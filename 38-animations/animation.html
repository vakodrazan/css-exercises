Keyframe Animation

  1. Keyframe Animation setup needs some unique code, that starts with @keyframe.
  2. What makes the keyframes code different to the rest of CSS only needs @keyframes and name.
  3. We can use a keyframe animation we set up earlier as a property in a ruleset for an element by using the specific name and call by using animation-name and animation-duration.
  4. Keyframe animations have other keywords that aren't available in `transitions`, these are keywords:
    • reverse
    • infinite
    • paused
    • running
    .
  5. We can animate as many properties as we want as long as the propperties are animatable at once.
  6. There are several of properties that are animatable, all the properties can change a value to one another.
