Overflow

  • Default value for each property

    Each property has their default value. Here they are:

    • overflow has its default value which is visible.
    • clip has a auto default value.
    • clip-path has a none default value.
    • text-overflow: ellipsis has a clip default value.

  • The reason of using those properties

    • We use overflow property because we want to specify a clip area or a scrollbar.
    • We use clip property because we want to make an absolute position of the elements.
    • We use clip-path property because it only specified the specific are of the elements but not all.
      • We use text-overflow: ellipsis; because it can cut off or hide the text which is overflow.
