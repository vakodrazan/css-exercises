Research about Custom properties

  1. We need double dashes (--) characters to define a Custom property's name.
  2. We cannot use space with the name.
  3. It depends on what value we want to set in with a new Custom property.
    • It is useful to create a Custom property :root or html when we don't want to repeat ourself for the same things.
    • Custom property isn't useful when we we want to limit the scoping.
  5. If we've defined a new Custom property we can recall it by using a var() function, but we don't repeat the value anymore when we use the var.
  6. We can override a Custom property by using:
    • --columns
    • --column-gap
    • --number-of-column
    • --base-unit
  7. Custom properties are so useful because it can help us saving our time when we want to edit the same thing.
  8. The only aspect of CSS syntax that we can use Custom properties for is element.
