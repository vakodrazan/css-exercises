CSS grid layout

Research about css grid layout

  • If we don't set any columns in in grid it will just try to fit in one column but if not, it'll create another column.
  • The difference between flexbox and grid:
    • Flexbox only affects one of rows or columns.
    • Grid affects both rows and columns at the same time.
  • When I want to specific what dimension I want to change I would use flexbox rather than grid because it is more specific.
  • I would use grid rather than flexbox because I want to change everything at the same time that means I want to change rows and columns at the same time without specified them.
  • We can have another way to write this grid-template-columns: 10rem 10rem 10rem 10rem;. This can be written as grid-template-columns: repeat(4, 10rem);.
  • We can specify a minimum height within its row increase in height if its content is too tall by using minmax() value.
