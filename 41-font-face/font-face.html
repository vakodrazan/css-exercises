Font face

  • There are several types of font formats, these are some of them:
    • Woff
    • Woff2
    • TTF
    • Otf
    • Eot
    • Svg
  • What I make of it woff is a good candidate to use in all browser.
  • Difference between a @font-face we have to use src: url() or local() and @keyframes / @media queries / @supports blocks
  • SOme fonts come with licencing restriction means they are free for everyone to use but it need to download.
  • If we want to support all the chosen fonts in here, we need to to provide eight font-face.
  • At roughly 20kb per font file, our users would have to download 160KB in the example above.
  • Most of our users would probably have the exact font we want to serve already installed on their devices. We can use the local(the font's name) to make sure that the user's browser doesn't download the fonts they might already have.
  • Sometimes we have to assign fallback fonts for our custom webfonts we add via @font-face.
  • The browser can use one of several strategies for loading the fonts, e.g. showing nothing until the font is downloaded, or using a fallback font first. We can use:
    • font-family
    • font-size
    • font-weight
    • font-style
