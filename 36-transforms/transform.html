Research about transform

  1. There are lots of function that we can use in a transform value. These are the list of them:
    • matrix()
    • perspective()
    • rotate()
    • scale()
    • skew()
    • translate()
  2. We have to use separate functions, in order, and as values, e.g. transform: scale(0.5) rotate(45deg); instead of using separate properties like this transform-scale: 0.5;, or transform-rotate: 45deg; because there is an order and relate on each other. There is no properties like transform-scale.
  3. If we want to move the central pin or axis of the transform away from its default position we can use the transform-origin property.
  4. It's preferable to use transform: translate(); rather than position and its associated properties because it is quick and easy.
    • When an element is translated or transformd it doesn't affect the flow of the document.
    • The layout positioning is similar to a transform.
  6. The problem you can run in to when overriding a transform in a more specific selector is the browser always create a style for the element that we create.
  7. If we want to go 3D, we need to use perspective property to set the view of the scene.
  8. We can control what the transform uses in its calculations in terms of the box model. We can use transform-box property to adjust this.
