Generated content, counters, and lists

In this exercise we are looking for generated content, counters, and lists

Pseudo-element

  • The special property for the pseudo-element is content. Without content property pseudo-element won't work.
  • The function which we can use in that property's value depends on what you want it to be.
  • These are the lists of the various pseudo-elements I could think:
    • ::before
    • ::after
    • ::first-letter
    • ::first-line
  • We use pseudo-elements because we don't need to specify it in HTML, we don't have to edit the HTML. Also, it can be specified the element.

Counters

  • We use pseudo-element to count in css.
  • There is some value that we can use when we want to reset a counter. We can use these values:
    • none
    • inherit
    • reset
    • unset
  • Counter are useful for the developer because they don't have to edit the html when they want to add another character lists.
