Positioning

Get to know more about position

Every property has their own default value. The default value of position is static.

CSS position

  • When use non-default position value we need to use one of these properties:
    • right
    • left
    • top
    • bottom
    • z-index
  • An element that has been positioned
  • position: relative; is a non-default value that still maintain a box in flow of a documents.
  • The special about fixed and sticky:
    • position: fixed; is positioned relative by the browser windows.
    • position: sticky; is based on the user's scroll position. It sticks in one place.
  • When we set an absolute value in elements but we don't want to coordinate the child with its parent element, we can set a position: inherit;.
  • Absolute positioned is useful for setting the position so we don't need to set any position for the child element anymore. All the child element will be positioned automatically.
  • When we put two absolutely positioned in elements that is in the physically space, it is only one will be visible because it seems as if they the same level. Also, it is always the latest one showed.
  • A CSS stacking context is a three-dimensional html element that has a special property.
