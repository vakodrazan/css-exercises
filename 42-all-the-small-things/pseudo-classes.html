Pseudo-classes

  • :focus is a pseudo-class used when an element is in a focus mode. This is well support for most of the browsers.
  • :focus-within is a type of pseudo-classes focus but the difference is that will focus all the within the container.
  • :visited is used for link that allows users to know that they have visited the link. It's color change when they click on it. Visited pseudo-class only work with the properties have a relation with coor.
  • :active is a pseudo pseudo-class. It is used to define when a user presses mouse down in the link or a paragraph
  • :target is used to highlight text that has been link to an element by using the id class to link it. It is useful when there is a long text but we want to search a specific deatil and we just link so the user will find it easily.
  • :disabled is used usually for a select element so when we select it, it can't activate anymore.
  • :empty is used for when there is nothing in one selector but we want to style it.
  • :not() is used when the element is not the element that we selected nor a child of that element.
  • :matches() is a fuctional pseudo-class help us to not repeat selector. Also, it allows us to write faster.
