Understanding all the litle detailes

Small Properties

  1. Cursor is used to define the character of the element when it hovers on the elements. there are different of type of cursor value we can have. For example: pointer, help, zoom-in, crosshair, text. There are a lot but it depends on the situation of what kind of element it is like link or text.
  2. Pointer-events allows us to click on something because if the pointer-event is none the pointer won't work for all the things.
  3. Outline is a shorthand property which allows us to draw a line outside the border. We can use different value in one property but we don't need comma to separate them like numeric color value but we need to put what kind of character it is such as solid or dashed.
  4. Filter is used for color. It's all about what type of lightness we want to use such as blur, contrast and drop-shadow.
  5. Appearance is to style specific element doesn't have it by defaul. It is not well support yet so we need the -webkit-, -moz- when we use it.
  6. Object-fit is used to specify for an image on how it recised. It will cut off the rest of the image if it's not fit the container. The background-size will have the value contain and cover.
  7. Scroll-snap-align
    8. is specifies the box's snap position ofr alignment of its snap area.
