Research about Shadows

  • box-shadow and text-shadow have a very similar value. However, text-shadow doesn't apply thickness and inset value whereas box-shadow does.
  • If we wanted to create a hard-edge shadow, one look like a solid border, we need zero for all the value but only the thickness we need a numeric value and color. For example: box-shadow: 0 0 0 1px blue;
  • It is possible to create a shadow on a circular element but polygon element can't only for svg can use a box-shadow.
  • It is possible to create a multiple shadows o the same element by using comma in between of them.
