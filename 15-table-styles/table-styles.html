Table styling

Here we are looking for the table elements.

Lists of properties associsted with table

  • The default value of table-layout is auto
  • The default value of border-spacing is number (2px).
  • The default value of border-collapse is separate.
  • The default value of caption-side is top or bottom.
  • The default value of empty-cells is show or hide.

Simulating HTML tables

We are looking for the html elements that matching with the css properties.

  • The table corresponds the table elements.
  • The table-row-group corresponds the tbody elements.
  • The table-header-group corresponds the thead elements.
  • The table-footer-group corresponds the tfoot elements.
  • The table-row corresponds the tr elements.
  • The table-cell corresponds the td and th elements.
  • The table-column-group corresponds the colgroup elements.
  • The table-column corresponds the col elements.
  • The table-caption corresponds the caption elements.

Further research

  • border-spacing: it can work effectively when we set the border-colapse: separate;.
  • The downside of using the various display table on non-table elements is sometimes we don't have the right the HTMl to use.
  • The difference betweeen table-layout: fixed; and table-layout: auto;.
    • table-layout: fixed; when you set width it doesn't affect with the columns but only the the rows.
    • table-layout: auto; when you set width it changed both rows and columns.
  • I think we can use all the properties like border-spacing on non-table elements if we use the right properties and values.
  • empty-cells can work effectively if we sets the border-colapse: separate;.
