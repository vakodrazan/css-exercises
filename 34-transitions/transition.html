Research about Transitions

  1. Transition property includes:
    • layouts property such as: width, height, padding, margins, borders
    • Positional properties like :left, right, topand bottom
    • Transformations
    • Font sizes
    • Colors
    • Background colors
    • Opacities
  2. The difference between timing, delay and duration are:
    • Timing is for how long does it take transition when it's on the way.
    • Delay is for how long does it have to stay before it starts its journey.
    • Duration is how long does it have to continues.

    • Text-enden
    • And also position.
  4. We can select more than one properties to transiton at a time by using pseudo-class hover and put all of the propertues that you want to enimate.
