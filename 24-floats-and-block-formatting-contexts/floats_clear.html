Learning floats, flow and formating contexts

Floats and clear properties

  1. Every property has their default value. We are looking for the default values for the float and clear in this research:
    • float: none;
    • clear: none;
  2. The possible values for clear are:
    • left
    • right
    • both
  3. The rule associated with a floated item in terms of source order is must come before of its content.
  4. We can always float other elements beside images when we emphasis it like right or left.
  5. We can create tow column layout with floats.

Block formating contexts

  • In my undestanding BFC is it can contain a float.
  • When we want to create a block formating context on a parents elements in order to clear a float we need some other properties with it, here they are:
    • position: absolute;
    • display: flow-root;
    • overflow
