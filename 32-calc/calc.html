CSS Calc function

  1. Calc function supports some the value that works in css such: number, integer, percentages, time, length, frequency.
  2. If we want to control the mathematical order of operation (PEDMAS / BODMAS) in a function we need to use brackets (...) with the value that we want to calculate like this calc((2+3) * 4) if we want to get 20 for the answer.
  3. We can convert one unit to another unit if we use multiplication and substraction.
  4. If we want to convert an unitless number into unit value we need to use calc function and mulplie the unitless number into the 1 and unit value it meant to be. This is an example: we want a rem value with the number 5, we need to do this calc(5 * 1rem).
  5. We need to put space around plus and minus operator when we use it inside a function because the if we don't put space the browser will expect that it is one value.
  6. We can convert a value into a negative value we need mulplie it by negative number like -1. For example: calc(5rem * -1).
  7. Calc function doesn't work with colour values.
